# Characters

## Creating a Character

A character has a name (2–24 characters) and a class. Class determines the character's starting stats.

## Classes and Base Stats

| Class   | Strength | Intelligence | Agility |
|---------|----------|--------------|---------|
| Warrior | 10       | 4            | 6       |
| Mage    | 3        | 12           | 5       |
| Rogue   | 6        | 6            | 11      |
| Ranger  | 7        | 5            | 9       |

## Stats

- **Strength** — influences physical activities and combat damage
- **Intelligence** — influences magic-based activities
- **Agility** — influences speed and precision-based activities (e.g. fishing catch chance)

## Storage

Characters are stored in memory for the lifetime of the server process. Restarting the server clears all data.
